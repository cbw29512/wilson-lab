# Demo Script (interview-ready)

## 2–3 minute demo flow
1) Open dashboard (GitHub Pages) — show clean UI, search, tag filters
2) Open a resource card — show description, tags, UTC timestamps
3) Log in as User — view-only capabilities
4) Log in as Admin — show extra actions + audit log visibility
5) Trigger a safe action (start/stop) and show audit entry

## Talking points
- Reproducible infrastructure via templates
- Security controls (RBAC + audit + confirmations)
- CI/CD and documentation discipline
