# Milestone M1 — React UI (mock inventory)

## Goal
Deliver a clean, professional-looking SaaS-style dashboard that lists lab resources (containers + VMs) with:
- Search box (matches name, description, tags)
- Tag filter dropdown
- Sort dropdown (most recent first)
- Resource cards with:
  - Type (Container/VM)
  - Status (Running/Stopped/Planned)
  - Created/Updated timestamps (UTC)
  - Short description
  - Tags
  - Buttons for View / Actions (wired later)

## What’s included
- React + TypeScript UI (frontend/)
- Mock inventory data rendered into cards
- Strong visual hierarchy for interview/demo readiness

## Local run
From repo root:
- `cd frontend`
- `npm install`
- `npm run dev -- --host 127.0.0.1 --port 5173`

## Acceptance checks
- UI loads cleanly with no console errors
- Search + tag filter + sort change the visible cards
- Timestamps render as UTC (ISO-8601)

## Next (M2)
- Replace mock inventory with real API inventory (Docker first, then VMs/libvirt)
- Add health checks + “delete/recreate” actions via backend
